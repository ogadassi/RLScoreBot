3-Step Setup Guide

Get your goal soundboard running in less than 60 seconds.

STEP 1

Add Bot to Discord

Click the Add to Discord button in the top navigation bar to invite RLScoreBot to your server.

STEP 2

Get Pairing Code

Type >link in your Discord server to generate your private 6-digit pairing key.

STEP 3

Paste in BakkesMod

Paste the code into the BakkesMod plugin menu in Rocket League. Upload custom sounds with >upload and score!
